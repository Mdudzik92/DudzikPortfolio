Let's Go!

Full Stack Web Development

  • React Expense Tracker

    This is an Expense Tracker app made with React in which the user can add to or deduct from their current amount with the use of functional components, hooks and the context API. The user can then remove transactions if needed

  • JS Drum Kit

    Vanilla JavaScript app in which the user can instantiate drum kit sounds as key events by simply pressing the keys on the screen. It plays the audio file associated with the key. The letters on the screen also highlight when clicked using CSS Transitions

  • Weather Tracker

    Weather app built with AJAX and a Web API from Openweather.org. The user can search by city to get the current temperature, humidity, wind speed, and UV index. The user also gets an upcoming 5-day forecast for their chosen city

  • Burger Menu

    Burger logger app created with Node, Express, MySQL, and Handlebars. The user can create a burger title for the menu, then add it to the 'devoured' list or the main menu. They can then choose to delete items from either menu

  • Home Scheduler

    MERN Stack calendar app in which a user can add the title, amount, and date due of a bill to a list. It then persists to their calendar. They can also do this with chores, and events, and also add items to grocery list

  • JS Text to Speech App

    The Voiceinator: the user can type their preferred message into the message input box, then have their message read back to them by whichever voice they choose from the dropdown menu. The user can also change the pitch and the speed of the message
